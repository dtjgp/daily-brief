# daily-brief

Automated daily brief from 4 sources:

1. Hacker News
2. GitHub Trending (proxy via GitHub search stars in last 7 days)
3. Product Hunt feed
4. X monitored stream (from local `X 日报` latest markdown)

## Run

```bash
npm run build
```

Outputs:
- `data/YYYY-MM-DD.json`
- `daily/YYYY-MM-DD.md`

## Notes

- Product Hunt feed is best-effort (public feed availability may vary).
- GitHub section currently uses a proxy method (new+starred repos in last 7 days).
- X section reads the latest local X report generated by your existing OpenClaw workflow.
